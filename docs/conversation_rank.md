# ConversationRank (ThinkerRank)

## Overview
ThinkerRank is implemented in `core/conversation_rank.js`.

It runs a weighted PageRank-style propagation over adjacency lists generated by `core/conversation_adjacency.js`.

## Graph Inputs
Nodes and edges come from `core/conversation_graph.js` / `core/thread_collapse.js`.

Propagation edge types:
- `reply` (`1.0`)
- `quote` (`1.3`)

## Formula
For node `i`:

```text
TR(i) = (1 - α) * base(i) + α * (incoming(i) + dangling_mass / N)
```

Where:

```text
incoming(i) = Σ_j [ TR(j) * w(j→i) / out_weight_sum(j) ]
```

Defaults:
- `α = 0.85`
- `minIterations = 10`
- `maxIterations = 20`
- `tolerance = 1e-6`
- `base(i) = 2.0` for followed authors, else `1.0`

## Determinism
Ranking output uses deterministic tie-breaking:
1. higher score
2. higher base score
3. earlier input index
4. lexical id

## Output

```js
{
  scores: [{ id, score, baseScore, inputIndex, tweet }],
  scoreById: Map,
  scoreByIdObject,
  topTweetIds,
  iterations,
  converged,
  scoreSpread,
  graphIndex
}
```

## Complexity
With `N` nodes and `E` edges:
- adjacency build: `O(N + E)`
- iterations: `O(k(N + E))`
- final sort: `O(N log N)`
